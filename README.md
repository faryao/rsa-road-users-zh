# RSA Road Users — English / 中文

A bilingual, Markdown-based reference mirror of the Road Safety Authority's Road Users section.

## Local development

```bash
npm install
npm run sync   # refresh source pages and Chinese translations
npm run build
npm start
```

Content is stored under `content/en` and `content/zh`. The deployed site includes full-text search and a persistent language switch.

> Unofficial reference project. Source content belongs to Ireland's Road Safety Authority. Always check the linked RSA page for current official guidance.
