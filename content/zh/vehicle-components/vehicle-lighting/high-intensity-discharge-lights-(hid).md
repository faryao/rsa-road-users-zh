# # 常见问题

[什么是高强度放电头灯 (HID) 和发光二极管 (LED) 头灯灯泡？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

|灯泡|描述 |
| ---| ---|
| HID 或氙气灯泡 |具有略带蓝色的色调，比普通卤素车头灯明亮得多。 |
| LED 灯泡 |其寿命和能源效率是卤素灯泡的数倍。LED 灯更集中，发出更明亮的光线，通常用于较新的汽车。 |

[在爱尔兰使用LED灯泡作为头灯合法吗？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

是的，一旦它们获得型式认证，安装到兼容车辆上，光束就不会令其他道路使用者眼花缭乱。

[改装HID头灯和LED灯泡合法吗？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

改装的 HID 和 LED 头灯灯泡（即非制造商安装的灯泡)并不总是与车辆或头灯兼容。请联系制造商或授权经销商。有关测试方法和失败原因，请参阅[NCT](http://www.rsa.ie/services/vehicle-owners/national-car-test-(nct)/about-the-national-car-test-(nct))或[CVRT 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.cvrt.ie/en-US/About-CVRT/New-CVR-Testing-Manuals/) 手册。

[道路适航性测试检查什么？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

适航性测试检查车辆灯光的对准情况，以确保它们处于水平状态。然而，驾驶员有责任确保其车辆适合行驶，因此定期检查所有车灯是否正常工作并正确对齐非常重要。

[如果我的车加装HID头灯会不会通过NCT？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

为了通过NCT：

* 改装的 HID 套件，即灯和灯泡必须经过型式认证
* 所安装的车辆必须兼容 HID 灯的使用，即车辆经批准可安装 HID 套件。
* 车辆的原始制造商将能够澄清您的车辆是否与 HID 套件兼容。

_将经批准的头灯套件改装到不兼容的车辆上将导致 NCT 失败。这同样适用于改装到兼容车辆上的未经批准的套件。将 HID 灯泡改装为常规卤素头灯也被列为 NCT 失败的原因之一。_

[什么是型式认证的 HID 系统？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)新车上的 HID 车头灯将在工厂安装并获得欧洲型式认证法规 (EU) 的批准 [2018/858 在新的浏览器选项卡中打开 在新的浏览器选项卡中打开](https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX:32018R0858&from=EN)，即，在欧洲销售之前，它们必须满足可接受的安全或质量标准。

此类 HID 系统包括专门设计用于容纳超亮 HID 灯泡的头灯、自动头灯调平和头灯清洁装置。前灯不水平或脏污会暂时使迎面而来的驾驶者失明并导致侧面视力不佳。

[汽车头灯标准有哪些规定？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

除了欧洲型式认证法规 (EU)

[2018/858 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX:32018R0858&from=EN) 外，它们在在欧洲销售之前还必须满足可接受的安全或质量标准。他们还必须遵守 [S.I. 1963 年第 189 号 在新浏览器选项卡中打开 在新浏览器选项卡中打开](http://www.irishstatutebook.ie/eli/1963/si/189/made/en/print) 1963 年《车辆道路交通照明条例》。

[我的汽车在出厂时就配备了 HID，但它们不能自动调平。 NCT会失败吗？](http://www.rsa.ie/road-safety/road-users/vehicle-components/vehicle-lighting/high-intensity-discharge-lights-(hid)#)

在某些情况下，特别是对于较旧的车辆，制造商安装了 HID 车头灯，但没有明显的手动或自动自调平装置。

如果原厂安装的HID头灯的车辆在悬架中内置了自动调平功能，那么可以认为这已经实现了头灯自动调平功能。

如果没有安装自动调平系统，车主必须出示原始制造商提供的证据，证明车辆在最初制造时配备了 HID 灯泡或前照灯，并且自动调平系统不属于本规范的一部分。
