# # 拖拉机巡视检查

[拖拉机巡视清单在新浏览器选项卡中打开](https://www.youtube.com/watch?v=zhl-BVhWwyI)您需要同意定位 cookie 才能加载此媒体[查看 Cookie 设置](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)

进行每日拖拉机巡视检查是帮助确保您和其他道路使用者安全的实用且有效的方法。现代拖拉机比以前更大、更快，因此如果不及早发现，即使是小问题也可能变得危险。通过花几分钟检查轮胎、灯光、制动器、连杆、液压、转向和能见度等部件，您可以在问题导致故障或碰撞之前发现它们，并帮助您的拖拉机延长使用寿命。将这些基本检查纳入您的日常工作是提高道路安全并保持拖拉机适航性的有效方法。

# # 农用拖拉机

农用拖拉机定义为：

* 至少有两个轴；
* 制造商规定的设计速度为6公里/小时或更高；
* 可配备一个或多个乘客座椅；
* 专门设计用于绘制和与其他设备附件配合使用，用于农业、渔业、林业或园艺工作。

# # 一般规则

## # 适航性测试

农用拖拉机不需要进行适航性测试，但法律要求您确保您的车辆在公共道路上使用时始终处于适航状态。

快速拖拉机必须经过适航性测试并出示适航证书 (CRW)。快速拖拉机是指最大设计速度超过40公里/小时的T类轮式拖拉机。

如果快速拖拉机仅在本州境内且主要在发生此类活动的土地上（包括农业道路、林业道路或农田）用于农业、园艺、林业、农业或渔业活动，或**仅在小岛上使用，则无需进行道路适航性测试。

请访问 CVRT 网站上的[快速拖拉机测试页面在新浏览器选项卡中打开](https://www.cvrt.ie/en-US/About-CVRT/Fast-Tractor-Testing/)了解更多详细信息。

## # 闪烁的琥珀色信标

所有拖拉机和自走式机器都必须安装闪烁的琥珀色信标，并在公共道路上运行时将其打开。

## # 车牌号

拖拉机的前部和后部必须安装车牌。请联系[收入在新浏览器选项卡中打开](https://www.revenue.ie/en/vrt/vehicle-registration-tax/registering-machinery.aspx)以获取更多信息。

## # 重量和尺寸

我们的[重量和尺寸](http://www.rsa.ie/road-safety/road-users/professional-drivers/vehicle-safety-legislation/weights-and-dimensions) 资料单概述了适用于农用拖拉机的法定宽度、长度、高度和重量限制。

## # 安全带

根据现行道路交通法，您不必在拖拉机上系安全带，但是，为了您自身的安全，我们始终建议您使用安全带。

# # 型式核准类别拖拉机按字母 T 分类，包括所有轮式拖拉机；每个轮式拖拉机类别根据其设计速度在“a”或“b”索引末尾补充：

“a”表示最大设计速度低于或等于 40 km/h 的轮式拖拉机，例如T1a、T2a

“b”代表最大设计速度超过 40 km/h 的轮式拖拉机，例如T1b、T2b

有关拖拉机类别的更多信息，请参阅[欧盟法规 167/2013 在新浏览器选项卡中打开](https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX%3A32013R0167&rid=6)

|拖拉机类别 |类别的定义|
| --- | --- |
| T1 |距离驾驶员最近的车轴的轮式拖拉机，最小履带宽度不小于 1,150 毫米，运行状态下空载质量超过 600 千克，离地间隙不大于 1,000 毫米 |
| T2 |最小履带宽度小于 1,150 毫米、空载运行质量超过 600 公斤的轮式拖拉机；离地间隙不大于600mm；如果拖拉机相对于地面的重心高度除以每个轴的平均最小轮距超过0.90，则最大设计速度应限制为30公里/小时。 |
| T3 |运行状态下空载质量不超过 600 公斤的轮式拖拉机 |
| T4 |最高设计车速不超过40公里/小时的专用轮式拖拉机 |
| T4.1 |高地隙拖拉机 |
| T4.2 |超宽拖拉机 |
| T4.3 |低净空拖拉机 |

# # 常见问题

[欧盟销售农用轮胎电子标记的法律是什么？](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)

农业或林业拖拉机、其拖车和可互换牵引机械及其系统、组件和单独技术单元的型式认证属于[爱尔兰国家标准局 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.nsai.ie/certification/automotive/national-type-approva/)(NSAI) 的职权范围。

[拖拉机法律是否要求进行走动检查，是否有检查表？](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)

不可以。车主有责任确保车辆始终适合行驶。请参阅 HSA“[基本拖拉机安全检查 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.hsa.ie/eng/publications_and_forms/publications/agriculture_and_forestry/a4_tractor_safety_checks_pdf_.pdf)”传单了解更多信息。

[如何获得拖拉机型式认证？](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)

您需要联系[爱尔兰国家标准局 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.nsai.ie/certification/automotive/national-type-approva/) (NSAI)，了解有关型式批准的信息。

您需要联系[收入 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.revenue.ie/en/contact-us/index.aspx) 询问在爱尔兰注册进口拖拉机需要哪些文件。

[拖拉机在道路上行驶时法律是否要求前挡泥板？](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)不可以。但是，根据 1993 年《道路法》，放置或存放“其拥有的动物的粪便或尿液，或者从其拥有或使用的车辆上掉落的任何材料或物品”均属违法行为。如需了解更多信息，请联系您的[地方当局 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.gov.ie/en/publication/942f74-local-authorities/)。

[我可以用农用拖拉机修剪边树篱吗？](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)

如果您在公共道路上使用农用拖拉机，则必须获得许可、纳税和投保。请联系负责树篱修剪和道路维护的[地方当局 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.gov.ie/en/publication/942f74-local-authorities/)。

[农用车公路使用前叉定位有何规定？](http://www.rsa.ie/road-safety/road-users/agricultural/agricultural-tractors#)

我们建议叉子或其他前置装卸附件应面朝下，因为它被认为是危险的突出物，会导致其他道路使用者受伤的风险。有关在农场安全使用拖拉机和机械的进一步指导，请参阅 [HSA 网站 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.hsa.ie/eng/Publications_and_Forms/Publications/Agriculture_and_Forestry/Guidance_on_the_Safe_Use_of_Tractors_and_Machinery_on_Farms.pdf)。

 是的。我们建议由具有适当资格的人员 (SQI) 按照制造商的规格进行安装，并满足 [(EU) 第 167/2013 号法规 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://eur-lex.europa.eu/legal-content/EN/TXT/PDF/?uri=CELEX:32013R0167&rid=2) 的基本技术规定。
