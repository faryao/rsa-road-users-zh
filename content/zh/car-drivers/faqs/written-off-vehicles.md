# 报废车辆

## # 有关报废车辆的信息和立法。

此内容仅供一般信息参考。它不会也无意提供法律或技术建议或代表其所解决问题的法律解释。

# # 常见问题

[爱尔兰报废车辆有哪些类别？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

* 注销 A 类 - 车辆被视为不适合修理。必须在不移除任何部件的情况下进行压碎。车辆无法返回道路。 （报废车辆）
* 类别注销 B - 车辆被视为不适合修理。可用部件可以回收。结构框架必须被压碎。车辆无法返回道路。 （报废车辆）
* 注销C类——车辆可修理；然而，维修费用超过了车辆事故前的价值。如果修好，车辆可以重新上路。
* 类别注销D - 车辆可修理；然而，出于经济原因，保险公司选择不修理车辆。在大多数情况下，如果送去维修，很可能会出现进一步的损坏。如果修好，车辆可以重新上路。

[谁负责将车辆送往授权治疗机构（ATF)？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

只有 A 类和 B 类注销需要由当前车辆所有权人提交给 ATF。

[如何查明车辆是否已被注销？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

即使是英国进口的车辆，您也可以通过在线网站付费查看车辆的历史记录。您还应该将其带到合格的机械师处，并就其是否处于适合道路行驶且安全驾驶的情况寻求建议。

[报废车辆如何通过NCT？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

如果通知 NCT，A 类和 B 类核销将不允许进行测试。 C 类和 D 类注销一旦修复即可重新上路，因此也可以进行测试。请务必记住，NCT 与保修不同。这是在某个时间点检查车辆是否符合基本标准。这不是详细的机械研究。 NCT 期间无法拆卸车辆，因此只能检查可见且可触及的部件。

[我私下买了车，发现被注销了，我有权利吗？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

您应该联系竞争和消费者保护委员会（[CCPC 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.ccpc.ie/))，以确定根据消费者保护立法您的权利和选择。此外，您不妨联系负责道路交通执法的警察局 (An Garda Síochána)。

[报废车辆可以登记吗？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

如果您的车辆属于 A 类或 B 类注销，您将无法再次注册。它必须被摧毁。如果是 C 或 D 类注销，您需要联系驾驶员和车辆计算机服务部门（[DVCSD 在新浏览器选项卡中打开 在新浏览器选项卡中打开](https://www.gov.ie/en/department-of-transport/policy-information/motor-tax-and-vehicle-ownership-nvdf/))或发送电子邮件至：[motortax@transport.gov.ie](mailto:motortax@transport.gov.ie) 以澄清需要采取什么行动。

[报废车辆可以投保吗？](http://www.rsa.ie/road-safety/road-users/car-drivers/resources/written-off-vehicles#)

任何保险查询都必须直接联系您的保险提供商。
